## 📊 核心功能模块

### 1. 历史记录系统
```javascript
// 自动记录用户操作
memoryStore.recordHistory('create_task', {
  taskId: 'task-001',
  title: '完成报告',
  type: 'report'
}, 'Tasks')
```

### 2. 智能推荐
```javascript
// 基于行为数据生成推荐
memoryStore.addRecommendation('frequentlyUsedTasks', {
  title: '周会准备',
  description: '您经常在周一创建会议任务',
  score: 85
})
```

### 3. 行为分析
```javascript
// 获取用户活跃时间模式
const activeHours = memoryStore.activeTimePattern
// 获取个性化推荐

```
